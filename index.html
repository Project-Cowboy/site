Cowboy: zkTLS for Cardano

What is Cowboy?

Cowboy is a decentralized protocol that can verify arbitrary web data while preserving privacy.

Why?

Web3 dapps need access to web data. Web3 apps require data from Web2—prices, posts, records—but can't trust it blindly. Cowboy uses TLSNotary-style verification to prove HTTPS sessions occurred, and offers selective redaction so only the necessary data is revealed.

How it works

  • Cowboy runs as a partnerchain that verifies TLSNotary data. As a partnerchain, it can do so optimally and configurably.
  • A decentralized notary set verifies zkTLS proofs.
  • Verified data is represented on-chain in some form.
  • Other apps or decentralized environments can reference verified Web2 facts.

Developer Example

Imagine a state website that provides access to a citizen-only portal. A zkTLS proof of authentication to this page can prove the user's unique citizenship. The user proves it privately, producing an on-chain credential that does not reveal their identity, but can be used in other dapps or off-chain applications.

Apply the same capability shown above to particular public or private web sources. TikTok, X, games, media, and you get an intuition of how the tool can be applied.
